# The Ultimate Job-Ready Data Science Course

A concise, hands-on Data Science course for beginners and intermediate learners — designed to make you job-ready with practical projects and real-world examples.

## What this repo contains
- **Course Overview:** A structured curriculum covering Python, data collection, NumPy, Pandas, visualization, SQL, probability, machine learning, deep learning, and web development for data scientists.
- **Notebooks & Examples:** Multiple Jupyter notebooks and example scripts for each module (see folder list below).
- **Projects:** Real projects including Coders of Delhi, Coders of Bangalore, and a RAG-based AI Teaching Assistant.

## Introduction
This course is a to-the-point, CodeWithHarry style Data Science curriculum that walks you from Python fundamentals to building ML models and production-ready RAG systems. It emphasizes hands-on learning through notebooks and projects.

## Course Modules
- Understanding the Conda Environment & Jupyter
- Python Refresher (data types, OOP, comprehensions, lambdas)
- NumPy: arrays, indexing, broadcasting
- Pandas: dataframes, cleaning, transformation, aggregation
- Visualization: Matplotlib & Seaborn
- Data Collection: requests, Beautiful Soup, HTML basics
- SQL for Data Science (MySQL basics, joins, views, stored procedures)
- Probability & Distributions
- Machine Learning with scikit-learn (pipelines, preprocessing, model evaluation)
- Deep Learning basics (TensorFlow / PyTorch overview)
- Web Development for Data Scientists (Flask APIs and templates)
- Large Language Models & RAG (embeddings, retrieval, building a RAG assistant)

## Projects
- Project 1 — Coders of Delhi: data cleaning, structuring, recommendations
- Project 2 — Coders of Bangalore: web scraping, parsing, analysis
- Project 3 — RAG-based AI Teaching Assistant: video transcription, chunking, embeddings, retrieval

## Requirements
Recommended: Anaconda (or Miniconda) with Python 3.8+ and the usual data science packages.

Create a conda environment and install common packages:

```bash
conda create -n ds-course python=3.10 -y
conda activate ds-course
pip install numpy pandas matplotlib seaborn scikit-learn jupyterlab requests beautifulsoup4 joblib tensorflow torch
```

## How to use this repository
- Open the notebooks with JupyterLab or VS Code.
- Run cells sequentially to reproduce analyses and experiments.
- Explore the `Data_Collection`, `Packages`, `Python`, and `Projects` folders for module-specific notebooks and data files.

## Notes
- No prior data science experience required — the course starts from fundamentals.
- Works best with a stable internet connection for external resources and package installs.
